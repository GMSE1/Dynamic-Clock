# Radiant Dynamic Clock

Here you will find a beautiful real-time clock with a gradiant blend of creme and sage.

## Technology Used

- React
- Vite
- date-fns
- CSS

## Prerequisties

Ensure your local computer has:
- Node.js
- npm (come installed with Node)

## Installation

1. Clone the repository or download the project files:

git clone git@github.com:GMSE1/Dynamic-Clock.git
cd color-clock

2. Install the dependencies:

npm install

## Running Locally

1. Start the development server:

npm run dev

2. Open your browser by bashing "o + enter"

3. You should see your dynamic color clock running!

## Customization

Want to change the colors or style? Edit the `src/App.css` file:
- Change the gradient colors in `.clock-container`
- Adjust font size in `.clock`
- Modify the glow effect in `text-shadow`
